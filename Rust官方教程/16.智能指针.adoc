== 简介

=== 指针(`pointer`)

* 是一个包含内存地址的变量的通用概念
* 除了引用数据没有任何其他特殊功能，也没有额外开销

=== 智能指针(`smart pointers`)

* 是一类数据结构，它们的表现类似指针，但是也拥有额外的元数据和功能
* 例如引用计数(`reference counting`)允许数据有多个所有者，它会记录所有者的数量，当没有所有者时清理数据
* 常见的智能指针有:
** `Box<T>`: 用于在堆上分配值
** `Rc<T>`: 一个引用计数类型，其数据可以有多个所有者
** `Ref<T>` 和 `RefMut<T>`: 通过 `RefCell<T>` 访问; `RefCell<T>` 是一个在运行时而不是在编译时执行借用规则的类型

== Box<T>
=== 简介
* 允许你将数据存储在堆上而非栈上; 留在栈上的是指向堆数据的指针
* 不会带来性能开销，但也没有很多额外功能

=== 应用场景
* 当你有一种类型，其大小在编译时无法确定，而你又想在需要确切大小的上下文中使用该类型的值时
* 当你拥有大量数据，并且想要转移所有权，但同时要确保在转移过程中数据不会被复制
* 当你想要拥有一个值，并且只关心它是实现了特定特征的类型，而不是特定的具体类型时

=== 使用
* 使用Box在堆上存储数据
+
[source,rust]
----
fn main() {
    let b = Box::new(5);
    println!("b = {b}");
}
----
** 我们可以像访问栈上的数据一样访问这个Box中的数据
** 就像任何拥有所有权的值一样，当一个Box超出作用域时，就像 b 在 main 末尾那样，它将被释放
** 释放操作既针对Box（存储在栈上），也针对它所指向的数据（存储在堆上）
