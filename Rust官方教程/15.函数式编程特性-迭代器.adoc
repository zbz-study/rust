== 迭代器定义

[source,rust]
----
pub trait Iterator {
    type Item;

    fn next(&mut self) -> Option<Self::Item>;

    // methods with default implementations elided
}
----

== 调用迭代器的 `next` 方法

[source,rust]
----
fn main() {
    let v1 = vec![1, 2, 3];

    let mut v1_iter = v1.iter();

    assert_eq!(v1_iter.next(), Some(&1));
    assert_eq!(v1_iter.next(), Some(&2));
    assert_eq!(v1_iter.next(), Some(&3));
    assert_eq!(v1_iter.next(), None);
}
----

* 必须将 v1_iter 设置为可变的，因为每次调用 `next` 都会从迭代器中移出一个元素
* `iter` 方法生成一个遍历 v1 的迭代器，调用 `next` 得到的值是对 v1 中值的不可变引用
* 如果想获取 v1 的所有权并返回拥有的值，应使用 `into_iter`
* 如果 v1 是可变的，应使用 `iter_mut`

== 在 `for` 循环中使用迭代器

+
[source,rust]
----
fn main() {
    let v1 = vec![1, 2, 3];

    let v1_iter = v1.iter();

    for val in v1_iter {
        println!("Got: {val}");
    }
}
----

* 不需要将 v1_iter 设置为可变的，因为 `for` 循环接管了 v1_iter 的控制权，并在幕后将其设置为可变的

== 创建另一个迭代器

[source,rust]
----
fn main() {
    let v1: Vec<i32> = vec![1, 2, 3];

    let v2: Vec<_> = v1.iter().map(|x| x + 1).collect();

    assert_eq!(v2, vec![2, 3, 4]);
}
----

* `map` 方法返回一个新的迭代器，该迭代器将调用 `x + 1` 的结果作为 `Item`
* `collect` 方法使用迭代器并将结果值收集到集合数据类型中
