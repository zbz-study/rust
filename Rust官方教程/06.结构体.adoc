== 定义结构体(Structs)

[source,rust]
----
struct User {
    active: bool,
    username: String,
    email: String,
    sign_in_count: u64,
}
----

== 创建结构体实例

[source,rust]
----
fn main() {
    let user1 = User {
        active: true,
        username: String::from("some user name123"),
        email: String::from("someone@example.com"),
        sign_in_count: 1,
    };
}
----

== 使用结构体实例

* 如果实例是可变的，可以更改字段的值，否则不能更改
+
[source,rust]
----
fn main() {
    let mut user1 = User {
        active: true,
        username: String::from("some user name123"),
        email: String::from("someone@example.com"),
        sign_in_count: 1,
    };

    user1.email = String::from("anotheremail@example.com");
}
----

== 字段初始化简写形式

[source,rust]
----
fn build_user(email: String, username: String) -> User {
    User {
        active: true,
        username,
        email,
        sign_in_count: 1,
    }
}
----

== 从一个现有的结构体创建新结构体

[source,rust]
----
fn main() {
    // --snip--

    let user2 = User {
        email: String::from("another@example.com"),
        ..user1
    };
}
----

[NOTE]
====
* 结构体更新语法使用 `=` 类似于赋值操作；因为它移动了数据(如 `user1` 的 `username` 被移动到了 `user2` 中)，所以创建 `user2` 之后，就不能再使用 `user1`
了
* 如果我们为 `user2` 的 `email` 和 `username` 都赋予了新的 `String` 值，而只使用了 `user1` 中的
`active` 和 `sign_in_count`，那么在创建 `user2` 之后，`user1` 仍然有效，因为 `active` 和 `sign_in_count` 都是实现了 `Copy`
特征的类型，而 `email` 和 `username` 没有被移出
====

== 元组结构体(Tuple Structs)

[source,rust]
----
struct Color(i32, i32, i32);
struct Point(i32, i32, i32);

fn main() {
    let black = Color(0, 0, 0);
    let origin = Point(0, 0, 0);
}
----

== 类单元结构体(Unit-Like Structs)

* 没有任何字段
* 当需要在某个类型上实现某个trait，但又不想在该类型中存储任何数据时，类单元结构体就会很有用

[source,rust]
----
struct AlwaysEqual;

fn main() {
    let subject = AlwaysEqual;
}
----

== 格式化输出结构体

* 使用派生Debug `{:?}`

[source,rust]
----
#[derive(Debug)]
struct Rectangle {
    width: u32,
    height: u32,
}

fn main() {
    let rect1 = Rectangle {
        width: 30,
        height: 50,
    };

    println!("rect1 is {rect1:#?}");    // 打印到标准输出控制台流（stdout）
    dbg!(&rect1);                       // 打印到标准错误控制台流（stderr）
}
----

== 方法

* 方法是定义在结构体或枚举类型中的函数
* 方法的第一个参数总是 `self`，表示方法所调用的实例(`&self` 是 `self: &Self`的简写)
* 方法的实现与函数一样，可以有返回值
* 方法与字段可以同名

[source,rust]
----
#[derive(Debug)]
struct Rectangle {
    width: u32,
    height: u32,
}

impl Rectangle {
    fn area(&self) -> u32 {
        self.width * self.height
    }
}

fn main() {
    let rect1 = Rectangle {
        width: 30,
        height: 50,
    };

    println!(
        "The area of the rectangle is {} square pixels.",
        rect1.area()
    );
}
----
